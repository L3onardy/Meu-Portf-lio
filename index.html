Calculadora Básica (Python)

Este projeto é uma Calculadora Básica desenvolvida em Python, utilizando a biblioteca customtkinter para criar uma interface gráfica moderna e amigável ao usuário. A calculadora permite realizar as quatro operações matemáticas fundamentais: Adição Subtração Multiplicação Divisão O principal objetivo do projeto foi praticar o uso da biblioteca customtkinter, explorando a criação de interfaces personalizadas com um visual mais moderno em comparação ao tkinter tradicional. A interface é simples, intuitiva e responsiva, ideal para usuários que buscam uma ferramenta básica e funcional
